# PRODUCT-DETAIL Intake

## 화면 목적

상품의 상세 정보를 확인하고, 옵션을 조합 선택하여 장바구니 담기 또는 바로 구매로 전환하는 페이지

## 핵심 행동

- 상품 이미지 확인 (다중 이미지 탐색)
- 상품 기본 정보 열람 (상품명, 가격, 할인, 배송 정보 등)
- 옵션 조합 선택 (사이즈, 컬러 등 복수 옵션 조합하여 선택)
- 수량 지정 및 선택 내역 확인
- 장바구니 담기
- 바로 구매
- 상세 설명 확인 (탭)
- 리뷰/평점 확인 (탭)
- 상품 문의 확인 (탭)

## 화면 구성

- 상단: 상품 이미지 갤러리 + 구매 영역 (상품 정보, 옵션 선택, 수량, 액션 버튼)
- 하단: 탭 구성 (상세정보 / 리뷰 / 문의)
- PC: 이미지와 구매 영역 좌우 배치
- 모바일: 이미지 → 구매 영역 수직 배치
