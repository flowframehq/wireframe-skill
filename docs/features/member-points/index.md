# 포인트/적립금

## 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 현재 잔액 | text | 보유 포인트 총액 |
| 포인트 이력 테이블 | table | 일시, 구분(적립/사용/조정), 금액, 사유 |
| 페이지네이션 | text | 페이지 이동 (최근 10건 단위) |

## 상태

| 상태 | 설명 |
|------|------|
| 로딩 | 포인트 정보 조회 중 |
| 표시 | 잔액 및 이력 정상 표시 |
| 빈 상태 | 포인트 이력 없음 |

## 인터랙션

- 페이지네이션으로 이전/다음 페이지 이동

## 비즈니스 로직

- 최신순 정렬 (기본)
- 잔액은 음수 불가
