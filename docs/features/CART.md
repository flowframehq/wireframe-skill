# 장바구니

## 장바구니 담기

### 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 장바구니 담기 버튼 | button | 선택된 옵션과 수량을 장바구니에 추가하는 버튼. 클릭 시 성공: 장바구니 담기 완료 안내 표시, 실패: 옵션 미선택 시 옵션 선택 안내 메시지 표시 |

### 상태

| 상태 | 설명 |
|------|------|
| 활성 | 옵션 선택이 완료되어 장바구니 담기가 가능한 상태 |
| 비활성 | 필수 옵션이 선택되지 않아 버튼이 비활성화된 상태 |
| 완료 | 장바구니 담기 성공 후 완료 안내가 표시된 상태 |
