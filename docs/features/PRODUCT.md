# 상품

## 상품 기본 정보

### 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 상품명 | text | 상품의 제목을 표시하는 텍스트 |
| 판매가 | text | 상품의 현재 판매 가격을 표시하는 텍스트 |
| 할인가 | text | 할인 적용 시 할인된 가격과 할인율을 함께 표시하는 텍스트. 할인이 없으면 미표시 |
| 배송 정보 | text | 배송비, 예상 배송일 등 배송 관련 안내를 표시하는 텍스트 |

### 상태

| 상태 | 설명 |
|------|------|
| 기본 | 상품 기본 정보가 정상 표시된 상태 |
| 품절 | 상품이 품절 상태임을 안내하는 표시 |

## 상품 이미지 갤러리

### 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 대표 이미지 | image | 현재 선택된 상품 이미지를 크게 표시하는 영역 |
| 썸네일 목록 | list | 상품의 전체 이미지를 작은 썸네일로 나열한 목록. 클릭 시 해당 이미지가 대표 이미지 영역에 표시 |

### 상태

| 상태 | 설명 |
|------|------|
| 기본 | 첫 번째 이미지가 대표 이미지로 선택된 상태 |
| 선택됨 | 사용자가 썸네일을 클릭하여 해당 이미지가 대표 이미지로 표시된 상태 |

### 인터랙션

- 썸네일 클릭 → 해당 이미지가 대표 이미지 영역에 표시

## 옵션 조합 선택

### 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 옵션 그룹 | select | 옵션 종류별(예: 컬러, 사이즈) 드롭다운 선택. 상품에 따라 1개 이상 표시 |
| 선택된 옵션 목록 | list | 사용자가 선택 완료한 옵션 조합을 표시하는 목록. 각 항목에 옵션명, 수량, 소계가 포함 |
| 옵션 삭제 버튼 | button | 선택된 옵션 목록에서 특정 옵션 조합을 제거하는 버튼. 클릭 시 해당 항목이 목록에서 삭제 |

### 상태

| 상태 | 설명 |
|------|------|
| 미선택 | 옵션을 아직 선택하지 않은 초기 상태. 선택된 옵션 목록 미표시 |
| 선택 완료 | 모든 필수 옵션을 선택하여 선택된 옵션 목록에 항목이 추가된 상태 |
| 품절 옵션 | 특정 옵션 조합이 품절인 경우 해당 옵션 비활성화 표시 |

### 인터랙션

- 옵션 그룹 선택 완료 → 선택된 옵션 조합이 선택된 옵션 목록에 추가
- 옵션 삭제 버튼 클릭 → 해당 옵션 조합이 목록에서 제거

## 수량 선택

### 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 수량 감소 버튼 | button | 선택된 옵션의 수량을 1 감소시키는 버튼. 수량이 1일 때 비활성화 |
| 수량 표시 | text | 현재 선택된 수량을 표시하는 텍스트 |
| 수량 증가 버튼 | button | 선택된 옵션의 수량을 1 증가시키는 버튼 |
| 합계 금액 | text | 선택된 모든 옵션의 수량과 단가를 합산한 총 금액을 표시하는 텍스트 |

### 인터랙션

- 수량 감소 버튼 클릭 → 수량 1 감소, 합계 금액 재계산
- 수량 증가 버튼 클릭 → 수량 1 증가, 합계 금액 재계산

## 상세 설명

### 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 상세 설명 콘텐츠 | text | 상품의 상세 설명(이미지, 텍스트 등)을 표시하는 콘텐츠 영역 |
