# 주문 메모

## 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 메모 목록 | list | 기존 메모 목록 (작성자, 작성일시, 내용) |
| 메모 입력 | input | 새 메모 내용 입력 |
| 등록 버튼 | button | 작성한 메모 저장 |

## 상태

| 상태 | 설명 |
|------|------|
| 메모 있음 | 기존 메모가 목록으로 표시 |
| 메모 없음 | 등록된 메모가 없을 때 빈 상태 안내 |

## 인터랙션

- 메모 입력 후 등록 버튼 클릭 → 메모 목록에 즉시 추가
- 메모 목록은 최신순 정렬

## 비즈니스 로직

- 메모는 관리자 내부용이며 고객에게 노출되지 않음
- 작성자는 현재 로그인한 관리자로 자동 지정
