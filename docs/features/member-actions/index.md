# 회원 관리 액션

## 공통 상태

| 상태 | 설명 |
|------|------|
| 비활성 | 회원 상세 미조회 상태, 모든 액션 버튼 비활성 |
| 활성 | 회원 상세 조회 완료, 액션 버튼 활성 |
| 처리 중 | 특정 액션 실행 중, 다른 액션 버튼 비활성 |

## 공통 인터랙션

- 모든 액션은 확인 모달을 거쳐 실행
- 액션 완료 시 성공/실패 토스트 메시지 표시
- 액션 완료 후 회원 기본 정보 영역 자동 갱신

## 공통 비즈니스 로직

- 차단된 회원에 대해서는 '차단 해제'만 활성, 나머지 액션 비활성
- 모든 액션은 처리자 정보와 함께 활동 로그에 기록
