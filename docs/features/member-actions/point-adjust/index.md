# 포인트 조정

## 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 포인트 조정 버튼 | button | 조정 모달 열기 |
| 구분 선택 | select | 적립 / 차감 |
| 금액 입력 | input | 조정할 포인트 금액 |
| 사유 입력 | input | 조정 사유 텍스트 |
| 확인 버튼 | button | 조정 실행 |
| 취소 버튼 | button | 모달 닫기 |

## 상태

| 상태 | 설명 |
|------|------|
| 닫힘 | 모달 미표시 |
| 입력 | 조정 정보 입력 모달 |
| 처리 중 | 조정 요청 처리 중 |

## 인터랙션

- 포인트 조정 버튼 클릭 → 모달 열림
- 확인 클릭 → 확인 모달 → 처리 → 잔액 및 이력 갱신
- 취소 클릭 → 모달 닫기

## 비즈니스 로직

- 금액: 양수만 입력 가능
- 차감 시 잔액 초과 불가
- 사유 필수 입력
- 차단/탈퇴 회원은 조정 불가
