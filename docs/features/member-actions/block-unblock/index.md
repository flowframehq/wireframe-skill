# 차단/해제

## 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 차단 버튼 | button | 활성 회원일 때 표시 |
| 차단 해제 버튼 | button | 차단 회원일 때 표시 |
| 사유 입력 | input | 차단/해제 사유 텍스트 |
| 확인 버튼 | button | 차단/해제 실행 |
| 취소 버튼 | button | 모달 닫기 |

## 상태

| 상태 | 설명 |
|------|------|
| 닫힘 | 모달 미표시 |
| 차단 입력 | 차단 사유 입력 모달 |
| 해제 입력 | 해제 사유 입력 모달 |
| 처리 중 | 차단/해제 요청 처리 중 |

## 인터랙션

- 차단 버튼 클릭 → 차단 사유 모달 → 확인 → 처리 → 회원 상태 갱신
- 차단 해제 버튼 클릭 → 해제 사유 모달 → 확인 → 처리 → 회원 상태 갱신
- 취소 클릭 → 모달 닫기

## 비즈니스 로직

- 사유는 필수 입력
- 차단 시 해당 회원 즉시 로그아웃 처리
- 탈퇴 회원은 차단/해제 불가
