# 정보 수정

## 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 정보 수정 버튼 | button | 수정 모달 열기 |
| 이름 입력 | input | 회원 이름 수정 |
| 연락처 입력 | input | 휴대폰 번호 수정 |
| 이메일 입력 | input | 이메일 주소 수정 |
| 저장 버튼 | button | 변경 사항 저장 |
| 취소 버튼 | button | 모달 닫기 |

## 상태

| 상태 | 설명 |
|------|------|
| 닫힘 | 모달 미표시 |
| 편집 중 | 모달 열림, 기존 값 채워진 상태 |
| 저장 중 | 변경 사항 저장 처리 중 |

## 인터랙션

- 정보 수정 버튼 클릭 → 모달 열림 (기존 값 채움)
- 저장 클릭 → 확인 모달 → 저장 처리 → 성공 시 모달 닫기 + 기본 정보 갱신
- 취소 클릭 → 모달 닫기

## 비즈니스 로직

- 연락처: 숫자만, 하이픈 자동 포맷
- 이메일: 형식 유효성 검사
- 차단/탈퇴 회원은 수정 불가
