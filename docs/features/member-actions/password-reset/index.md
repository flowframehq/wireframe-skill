# 비밀번호 초기화

## 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 비밀번호 초기화 버튼 | button | 초기화 확인 모달 열기 |
| 안내 텍스트 | text | 초기화 시 회원 이메일로 재설정 링크 발송 안내 |
| 확인 버튼 | button | 초기화 실행 |
| 취소 버튼 | button | 모달 닫기 |

## 상태

| 상태 | 설명 |
|------|------|
| 닫힘 | 모달 미표시 |
| 확인 | 초기화 확인 모달 |
| 처리 중 | 초기화 요청 처리 중 |
| 완료 | 발송 완료 안내 |

## 인터랙션

- 비밀번호 초기화 버튼 클릭 → 확인 모달
- 확인 클릭 → 처리 → 완료 메시지 표시
- 취소 클릭 → 모달 닫기

## 비즈니스 로직

- 회원 이메일로 비밀번호 재설정 링크 발송
- 차단/탈퇴 회원은 초기화 불가
- 재설정 링크 유효기간 24시간
