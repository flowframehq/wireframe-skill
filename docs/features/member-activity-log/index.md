# 활동 로그

## 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 활동 로그 테이블 | table | 일시, 활동 유형, 상세 내용, IP 주소 |
| 페이지네이션 | text | 페이지 이동 (최근 20건 단위) |

## 상태

| 상태 | 설명 |
|------|------|
| 로딩 | 로그 조회 중 |
| 표시 | 로그 목록 정상 표시 |
| 빈 상태 | 활동 기록 없음 |

## 인터랙션

- 페이지네이션으로 이전/다음 페이지 이동

## 비즈니스 로직

- 최신순 정렬
- 활동 유형: 로그인, 정보 변경, 주문, 관리자 액션 등 자동 기록
- 로그는 삭제 불가 (감사 추적용)
