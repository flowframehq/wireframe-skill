# 주문

## 바로 구매

### 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 바로 구매 버튼 | button | 선택된 옵션과 수량으로 즉시 주문 페이지로 이동하는 버튼. 클릭 시 성공: 주문서 작성 화면으로 이동, 실패: 옵션 미선택 시 옵션 선택 안내 메시지 표시 |

### 상태

| 상태 | 설명 |
|------|------|
| 활성 | 옵션 선택이 완료되어 바로 구매가 가능한 상태 |
| 비활성 | 필수 옵션이 선택되지 않아 버튼이 비활성화된 상태 |
