# 주문 목록

## 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 검색어 입력 | input | 주문번호, 주문자명, 상품명으로 검색 |
| 기간 필터 | input | 시작일~종료일 범위 선택 |
| 주문상태 필터 | select | 전체/신규주문/확인/처리중/완료/취소 |
| 배송상태 필터 | select | 전체/준비중/배송중/배송완료/반송 |
| 검색 버튼 | button | 입력한 조건으로 주문 목록 조회 |
| 초기화 버튼 | button | 검색 조건을 기본값으로 되돌리기 |
| 주문 테이블 | table | 주문번호, 주문일시, 주문자명, 대표상품(외 N건), 결제금액, 주문상태, 배송상태 |
| 페이지네이션 | list | 페이지 번호 이동 |

## 상태

| 상태 | 설명 |
|------|------|
| 기본 | 최근 주문 기준 목록 표시 |
| 검색 결과 | 필터/검색 조건 적용된 결과 표시 |
| 결과 없음 | 조건에 맞는 주문이 없을 때 빈 상태 안내 |
| 로딩 | 주문 목록 조회 중 |

## 인터랙션

- 검색 조건 입력 후 검색 버튼 클릭 → 조건에 맞는 주문 목록 조회
- 초기화 버튼 클릭 → 모든 필터/검색어 초기화 후 전체 목록 표시
- 주문 행 클릭 → 해당 주문의 상세 정보 표시
- 페이지 번호 클릭 → 해당 페이지 주문 목록 표시

## 비즈니스 로직

- 기본 정렬: 주문일시 최신순
- 페이지당 표시 건수: 기본 20건
- 대표상품은 주문 내 첫 번째 상품명 + "외 N건" 형태로 요약 표시
