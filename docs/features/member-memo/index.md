# 관리자 메모

## 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 메모 입력란 | input | 새 메모 작성 텍스트 영역 |
| 등록 버튼 | button | 메모 저장 |
| 메모 목록 | list | 작성일시, 작성자, 내용 |

## 상태

| 상태 | 설명 |
|------|------|
| 로딩 | 메모 목록 조회 중 |
| 표시 | 메모 목록 정상 표시 |
| 빈 상태 | 등록된 메모 없음 |

## 인터랙션

- 메모 작성 후 등록 버튼 클릭 → 목록에 추가
- 본인 작성 메모 삭제 가능

## 비즈니스 로직

- 최신순 정렬
- 메모는 관리자만 조회 가능 (회원에게 비공개)
- 삭제는 본인 작성분만 가능
