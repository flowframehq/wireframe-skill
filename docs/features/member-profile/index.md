# 회원 기본 정보

## 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 회원 ID | text | 시스템 고유 회원번호 |
| 이름 | text | 회원 이름 |
| 연락처 | text | 휴대폰 번호 |
| 이메일 | text | 이메일 주소 |
| 가입일 | text | 가입 일시 |
| 회원 상태 | text | 활성 / 차단 / 탈퇴 |
| 최근 접속일 | text | 마지막 로그인 일시 |
| 누적 구매금액 | text | 총 구매 금액 |

## 상태

| 상태 | 설명 |
|------|------|
| 로딩 | 회원 상세 데이터 불러오는 중 |
| 표시 | 회원 정보 정상 표시 |

## 인터랙션

- 페이지 진입 시 회원 ID 기반으로 상세 정보 조회
- 관리 액션(정보 수정, 차단/해제 등) 완료 시 자동 갱신

## 비즈니스 로직

- 읽기 전용 영역, 수정은 '정보 수정' 액션을 통해서만 가능
- 탈퇴 회원은 개인정보 마스킹 처리 (이름, 연락처, 이메일)
