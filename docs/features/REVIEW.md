# 리뷰

## 평점 요약

### 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 평균 평점 | text | 전체 리뷰의 평균 별점을 숫자와 별 아이콘으로 표시하는 텍스트 |
| 총 리뷰 수 | text | 등록된 전체 리뷰 건수를 표시하는 텍스트 |
| 별점 분포 | list | 1점부터 5점까지 각 별점별 리뷰 건수를 막대 그래프로 표시하는 분포 목록 |

## 리뷰 목록 조회

### 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 리뷰 정렬 | select | 최신순, 평점 높은순, 평점 낮은순 등으로 리뷰 목록을 정렬하는 드롭다운 |
| 리뷰 목록 | list | 작성자, 별점, 리뷰 내용, 작성일을 포함한 개별 리뷰 항목의 목록 |
| 더보기 버튼 | button | 추가 리뷰를 불러오는 버튼. 클릭 시 다음 페이지의 리뷰가 목록 하단에 추가 표시 |

### 상태

| 상태 | 설명 |
|------|------|
| 기본 | 리뷰 목록이 표시된 상태 |
| 빈 상태 | 등록된 리뷰가 없을 때 리뷰 없음 안내 메시지 표시 |
| 전체 로드 완료 | 모든 리뷰를 불러와 더보기 버튼이 미표시된 상태 |

### 인터랙션

- 리뷰 정렬 변경 → 선택한 기준으로 리뷰 목록 재정렬
- 더보기 버튼 클릭 → 다음 페이지의 리뷰를 목록 하단에 추가 로드
