# 결제 수단

## 공통 상태

| 상태 | 설명 |
|------|------|
| 미선택 | 결제 수단이 선택되지 않은 초기 상태 |
| 선택됨 | 하나의 결제 수단이 선택되어 해당 입력 폼 표시 |
| 입력 완료 | 선택한 결제 수단의 필수 정보 입력 완료 |

## 공통 인터랙션

- 결제 수단 탭 클릭 → 해당 결제 수단 입력 폼으로 전환, 이전 입력 내용 유지
- 하나의 결제 수단만 선택 가능 (라디오 방식)

## 공통 비즈니스 로직

- 결제 수단 미선택 시 결제 버튼 비활성화
- 선택한 결제 수단의 필수 정보가 모두 입력되어야 결제 진행 가능
