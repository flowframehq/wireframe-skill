# 주문/구매 이력

## 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 주문 이력 테이블 | table | 주문번호, 주문일시, 상품명, 결제금액, 주문상태 |
| 전체보기 링크 | link | 주문관리 화면으로 이동 (해당 회원 필터 적용) |
| 페이지네이션 | text | 페이지 이동 (최근 10건 단위) |

## 상태

| 상태 | 설명 |
|------|------|
| 로딩 | 주문 이력 조회 중 |
| 목록 표시 | 주문 이력 정상 표시 |
| 빈 상태 | 주문 이력 없음 |

## 인터랙션

- 주문번호 클릭 → 주문관리 상세 화면으로 이동
- 전체보기 클릭 → 주문관리 화면으로 이동 (해당 회원 필터)
- 페이지네이션으로 이전/다음 페이지 이동

## 비즈니스 로직

- 최신순 정렬 (기본)
- 탈퇴 회원도 주문 이력은 조회 가능
