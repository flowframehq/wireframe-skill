# 주문 상세

## 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 주문번호 | text | 선택된 주문의 고유 번호 |
| 주문일시 | text | 주문이 접수된 일시 |
| 주문상태 | text | 현재 주문 상태 표시 |
| 주문자 정보 | text | 주문자 이름, 연락처, 이메일 |
| 수령자 정보 | text | 수령자 이름, 연락처, 배송지 주소 |
| 상품 목록 | table | 상품명, 옵션, 수량, 단가, 소계 |
| 결제 정보 | text | 결제수단, 결제금액, 할인금액, 최종결제금액 |
| 처리 이력 | list | 주문 상태 변경 이력 (일시, 상태, 처리자) |

## 상태

| 상태 | 설명 |
|------|------|
| 미선택 | 주문이 선택되지 않은 초기 상태, 빈 영역 표시 |
| 조회 중 | 주문 상세 데이터 로딩 |
| 상세 표시 | 선택된 주문의 전체 정보 표시 |

## 인터랙션

- 주문 목록에서 행 클릭 → 상세 영역에 해당 주문 정보 표시
- 상세 영역 내 주문 상태/배송 상태 → 액션 버튼 활성화 상태에 반영

## 비즈니스 로직

- 주문 상세는 읽기 전용, 수정은 주문 처리 액션을 통해서만 가능
- 처리 이력은 최신순으로 정렬
