# 상품 문의

## 문의 목록 조회

### 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 문의 목록 | list | 작성자, 문의 제목, 작성일, 답변 여부를 포함한 문의 항목의 목록. 비밀글인 경우 제목이 마스킹 처리되어 표시 |
| 문의 상세 토글 | button | 문의 항목 클릭 시 해당 문의의 본문과 답변을 펼쳐 보거나 접는 토글 버튼 |
| 더보기 버튼 | button | 추가 문의를 불러오는 버튼. 클릭 시 다음 페이지의 문의가 목록 하단에 추가 표시 |

### 상태

| 상태 | 설명 |
|------|------|
| 기본 | 문의 목록이 표시된 상태 |
| 빈 상태 | 등록된 문의가 없을 때 문의 없음 안내 메시지 표시 |
| 펼침 | 특정 문의 항목이 펼쳐져 본문과 답변이 표시된 상태 |
| 전체 로드 완료 | 모든 문의를 불러와 더보기 버튼이 미표시된 상태 |

### 인터랙션

- 문의 항목 클릭 → 해당 문의의 본문과 답변 내용을 펼침/접기 토글
- 더보기 버튼 클릭 → 다음 페이지의 문의를 목록 하단에 추가 로드
