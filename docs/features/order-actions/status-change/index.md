# 상태 변경

## 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 상태 선택 | select | 변경할 주문 상태 선택 |
| 변경 사유 | input | 상태 변경 사유 입력 (선택) |
| 상태 변경 버튼 | button | 선택한 상태로 변경 실행 |

## 상태

| 상태 | 설명 |
|------|------|
| 변경 가능 | 현재 주문 상태에서 전환 가능한 상태 목록 표시 |
| 변경 불가 | 전환 가능한 상태가 없어 비활성화 |

## 인터랙션

- 상태 선택 후 변경 버튼 클릭 → 확인 다이얼로그 표시
- 확인 → 상태 변경 실행, 주문 상세와 목록 갱신
- 취소 → 변경 취소, 이전 상태 유지

## 비즈니스 로직

- 전환 가능한 상태는 현재 상태에 따라 결정 (상태 머신)
- 상태 변경 시 처리 이력에 자동 기록
