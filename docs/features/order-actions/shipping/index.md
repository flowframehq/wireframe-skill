# 배송 처리

## 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 배송사 선택 | select | CJ대한통운, 한진택배, 롯데택배, 우체국택배, 로젠택배 등 |
| 송장번호 입력 | input | 송장번호 입력 필드 (숫자만, placeholder: "송장번호를 입력하세요") |
| 배송 등록 버튼 | button | 배송사 + 송장번호 저장 및 "배송중" 상태 전이 |
| 배송 정보 표시 | text | 등록된 배송사, 송장번호, 등록일시 표시 |
| 배송 추적 링크 | link | 배송사 추적 페이지로 이동하는 외부 링크 |

## 상태

| 상태 | 설명 |
|------|------|
| 미등록 | 배송준비 상태, 배송사/송장번호 입력 폼 표시 |
| 등록완료 | 배송중/배송완료 상태, 등록된 배송 정보 + 추적 링크 표시 |
| 입력 오류 | 송장번호 미입력 또는 형식 오류 시 유효성 안내 |
| 처리 중 | 등록 버튼 비활성화 + 스피너 |

## 인터랙션

- 배송사 선택 + 송장번호 입력 후 등록 버튼 클릭 → 배송 정보 저장, 주문 상태 "배송중"으로 자동 전이
- 배송 추적 링크 클릭 → 새 탭에서 배송사 추적 페이지 열림
- 송장번호 미입력 시 등록 버튼 클릭 → 유효성 오류 표시

## 유저스토리

- 쇼핑몰 관리자로서 송장번호를 등록하고 싶다, 고객에게 배송 추적 정보를 제공하기 위해
- 쇼핑몰 관리자로서 배송 현황을 빠르게 확인하고 싶다, 배송 문의에 응대하기 위해

## 인수조건

- Given 배송준비 상태 주문에서 When 배송사와 송장번호를 입력하고 등록하면 Then 주문 상태가 "배송중"으로 변경된다
- Given 송장번호를 입력하지 않고 When 등록 버튼을 클릭하면 Then 유효성 오류 메시지가 표시된다
- Given 배송 정보가 등록된 주문에서 When 배송 추적 링크를 클릭하면 Then 배송사 추적 페이지가 새 탭에서 열린다
- Given 배송중 상태 주문에서 When 배송 처리 영역을 보면 Then 등록된 배송사, 송장번호, 추적 링크가 표시된다

## 비즈니스 로직

- 배송 등록 시 주문 상태가 자동으로 "배송중"으로 전이
- 송장번호는 숫자만 허용, 최소 10자 이상
- 배송사별 추적 URL 패턴에 송장번호를 삽입하여 추적 링크 생성
- 배송 정보 등록 후 수정 가능 (송장번호 오입력 대응)
