# 배송 처리

## 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 택배사 선택 | select | 배송 택배사 선택 |
| 송장번호 입력 | input | 배송 송장번호 입력 |
| 배송 등록 버튼 | button | 배송 정보 등록 실행 |

## 상태

| 상태 | 설명 |
|------|------|
| 등록 가능 | 주문 확인 완료 상태에서 배송 정보 입력 가능 |
| 등록 불가 | 주문 상태가 배송 등록 조건을 충족하지 않음 |
| 등록 완료 | 배송 정보가 이미 등록된 상태 (수정 가능) |

## 인터랙션

- 택배사 선택 + 송장번호 입력 후 등록 버튼 클릭 → 배송 정보 저장
- 등록 완료 후 → 배송상태 '배송중'으로 자동 변경, 주문 상세와 목록 갱신

## 비즈니스 로직

- 택배사와 송장번호 모두 필수 입력
- 배송 등록 시 주문 상태가 '처리중' 이상이어야 함
- 배송 등록 후에도 송장번호 수정 가능
