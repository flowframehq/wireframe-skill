# 취소/환불

## 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 처리 유형 | select | 취소 또는 환불 선택 |
| 사유 선택 | select | 취소/환불 사유 선택 |
| 상세 사유 | input | 추가 사유 상세 입력 (선택) |
| 환불 금액 | text | 환불 예정 금액 표시 |
| 실행 버튼 | button | 취소/환불 처리 실행 |

## 상태

| 상태 | 설명 |
|------|------|
| 취소 가능 | 배송 전 상태에서 주문 취소 가능 |
| 환불 가능 | 배송 완료 후 환불 처리 가능 |
| 처리 불가 | 이미 취소/환불 처리된 주문 |

## 인터랙션

- 처리 유형 + 사유 선택 후 실행 버튼 클릭 → 확인 다이얼로그 표시
- 확인 → 취소/환불 실행, 주문 상태와 결제 상태 갱신
- 취소 → 이전 상태 유지

## 비즈니스 로직

- 배송 전 → 취소만 가능, 배송 완료 후 → 환불만 가능
- 환불 금액은 결제 금액 기준으로 자동 계산 (부분 환불 여부는 정책에 따름)
- 취소/환불 처리 시 결제 취소 또는 환불 프로세스 연동
