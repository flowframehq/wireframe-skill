# 회원 관리

## 회원 목록

### 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 검색 입력 | input | 이름 또는 이메일로 회원을 검색하는 입력 필드 |
| 상태 필터 | select | 활성/정지/탈퇴 상태로 회원 목록을 필터링하는 드롭다운 |
| 역할 필터 | select | 관리자/일반/VIP 역할로 회원 목록을 필터링하는 드롭다운 |
| 회원 테이블 | table | 이름, 이메일, 가입일, 상태, 역할을 표시하는 회원 목록 테이블. 행 클릭 시 우측 상세 패널에 해당 회원 정보 표시 |
| 페이지네이션 | button | 이전/다음 페이지 이동 및 페이지 번호 선택 버튼 |

### 상태

| 상태 | 설명 |
|------|------|
| 기본 | 회원 목록이 표시된 상태 |
| 빈 상태 | 검색/필터 조건에 맞는 회원이 없을 때 안내 메시지 표시 |

### 인터랙션

- 검색 입력 → 이름 또는 이메일 기준으로 목록 필터링
- 상태 필터 변경 → 선택한 상태의 회원만 표시
- 역할 필터 변경 → 선택한 역할의 회원만 표시
- 회원 행 클릭 → 우측 상세 패널에 해당 회원 정보 로드

## 회원 상세

### 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 프로필 정보 | text | 회원 이름, 이메일, 가입일, 현재 상태, 현재 역할을 표시하는 읽기 전용 영역 |
| 메모 | textarea | 관리자가 회원에 대한 메모를 작성하는 인라인 편집 가능한 텍스트 영역. 클릭 시 편집 모드로 전환, 포커스 아웃 시 자동 저장 |
| 상태 변경 버튼 | button | 클릭 시 상태 변경 확인 모달을 열어 회원 상태를 변경하는 버튼. 본인 계정일 경우 비활성화 |
| 역할 변경 버튼 | button | 클릭 시 역할 변경 모달을 열어 회원 역할을 부여/해제하는 버튼 |

### 상태

| 상태 | 설명 |
|------|------|
| 미선택 | 좌측 목록에서 회원을 선택하지 않은 초기 상태. 안내 메시지 표시 |
| 선택됨 | 회원이 선택되어 프로필, 메모, 액션 버튼이 표시된 상태 |

### 인터랙션

- 메모 필드 클릭 → 인라인 편집 모드 활성화
- 메모 포커스 아웃 → 변경 사항 자동 저장
- 상태 변경 버튼 클릭 → 상태 변경 확인 모달 오픈
- 역할 변경 버튼 클릭 → 역할 변경 모달 오픈

## 상태 변경

### 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 현재 상태 | text | 변경 대상 회원의 현재 상태를 표시하는 텍스트 |
| 변경할 상태 | select | 활성/정지/탈퇴 중 변경할 상태를 선택하는 드롭다운 |
| 변경 사유 | textarea | 상태 변경 사유를 입력하는 필수 텍스트 영역 |
| 확인 버튼 | button | 입력된 사유와 함께 상태 변경을 확정하는 실행 버튼. 사유 미입력 시 비활성화 |
| 취소 버튼 | button | 상태 변경을 취소하고 모달을 닫는 버튼 |

### 상태

| 상태 | 설명 |
|------|------|
| 기본 | 사유 미입력, 확인 버튼 비활성화 |
| 입력 완료 | 사유 입력 완료, 확인 버튼 활성화 |

### 비즈니스 로직

- 상태 변경 시 사유 필수 입력
- 본인 계정은 상태 변경 불가 (상세 패널에서 버튼 비활성화로 처리)

## 역할 변경

### 와이어프레임 요소

| 요소 | type | 설명 |
|------|------|------|
| 현재 역할 | text | 변경 대상 회원의 현재 역할을 표시하는 텍스트 |
| 변경할 역할 | select | 관리자/일반/VIP 중 변경할 역할을 선택하는 드롭다운 |
| 확인 버튼 | button | 역할 변경을 확정하는 실행 버튼 |
| 취소 버튼 | button | 역할 변경을 취소하고 모달을 닫는 버튼 |

### 상태

| 상태 | 설명 |
|------|------|
| 기본 | 현재 역할이 표시되고 변경할 역할을 선택할 수 있는 상태 |

### 인터랙션

- 역할 선택 → 확인 버튼 클릭 시 역할 변경 반영
